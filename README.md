# Daily BMI Calculator 2026-04-30

## Description

A lightweight Python BMI (Body Mass Index) calculator tool designed for daily health tracking. This application provides quick and accurate BMI calculations based on height and weight inputs, helping users monitor their body composition metrics.

## Features

- **Quick BMI Calculation** - Calculate BMI from height and weight inputs
- **Multiple Unit Support** - Works with both metric (kg, cm) and imperial (lbs, inches) units
- **Health Category Classification** - Automatically categorizes results (Underweight, Normal, Overweight, Obese)
- **User-Friendly Interface** - Simple command-line or GUI-based interaction
- **Input Validation** - Ensures accurate and valid data entry
- **Daily Tracking** - Log and track BMI measurements over time

## How to Run

### Prerequisites

- Python 3.7 or higher
- pip (Python package manager)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd daily-BMI-calculator-2026-04-30
   ```

2. Install dependencies (if any):
   ```bash
   pip install -r requirements.txt
   ```

### Usage

Run the calculator:

```bash
python bmi_calculator.py
```

Follow the on-screen prompts to:
1. Select your preferred unit system (metric/imperial)
2. Enter your height
3. Enter your weight
4. View your BMI result and health category
